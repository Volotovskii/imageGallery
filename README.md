# Заготовка учебного проекта для практики Модуль 50. Работа с БД

## Описание проекта

В качестве практической работы мы создадим галерею изображений со следующими особенностями:

1.  Используется роутинг (маршрутизация) в файле `index.php`.
2.  JavaScript-файлы подключены в виде модулей. Главный файл: `main.js`.
3.  Фотографии сохраняются на сервере под уникальным именем.
4.  Единый CSS-файл для всего проекта.
5.  Применена архитектура MVC.
6.  Необходимо создать таблицы в базе данных по прилагаемой инструкции, так как используется механизм `CASCADE` для MySQL (привязка фотографии к комментариям).

### Примечание по базе данных:

Для таблиц `comments` и `images` обязательно укажите `ENGINE=InnoDB`.

## Конфигурация базы данных

Доступы к базе данных указаны в файле `app/config/config.php`. При создании своей БД используйте те же параметры (или измените их в конфиге).

**Важно:** Подробная инструкция по созданию базы данных и таблиц находится в файле `db_config.txt`.

## Среда развертывания (Использовалось при разработке)

Проект был развернут в Wampserver со следующими версиями:

1.  **Apache:** 2.4.62 (Win64) PHP/8.3.14 mod_fcgid/2.3.10-dev
2.  **MySQL:** 9.1.0

### Инструкция по развертыванию в Wampserver:

После установки Wampserver, перейдите в его корневую директорию (например, `C:\wamp64\www`) и клонируйте репозиторий проекта туда.

## Доступ к проекту

После успешного развертывания, проект будет доступен по следующему URL:
`http://localhost/imageGallery/public/`
